### 数据分类

- 文章数据
  - title: 标题
  - date：文章创建时间
  - lastDate: 最后编辑时间
  - tags: 标签
  - readCount：阅读量
  - abstract：摘要
  - content：内容
  - mdContent：内容的 md 格式
  - type: 是否私密（发布草稿）
